Field Lab

AI-powered environmental data analysis and visualization.

App coming soon

Features

  • Load environmental datasets (e.g., ADB indicators)
  • Run correlation and AI-assisted analyses
  • Export and visualize results

Run locally

  1. Create a virtual environment and install dependencies:
uv sync
# or
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
  1. Start the Streamlit app:
streamlit run app.py

GitHub Pages serves static files only. For the interactive app, deploy to Streamlit Cloud, Hugging Face Spaces, Render, etc., and link it above.
